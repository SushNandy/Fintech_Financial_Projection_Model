# Fintech_Financial_Projection_Model
📌 Project Overview
This project is a 3-year financial projection model developed for a fintech lending company using Microsoft Excel. The model forecasts revenue, operating costs, profitability, and key financial metrics for FY2024–FY2026 based on dynamic business assumptions.

The model is fully formula-driven and updates automatically when the selected business scenario changes (Conservative, Moderate, or Aggressive).

🎯 Objective
Build a dynamic financial projection model.
Forecast company performance for the next three financial years.
Eliminate manual calculations using Excel formulas.
Enable scenario-based financial planning and decision-making.
